WSP Design System

Tooltip

A tooltip is often used to specify extra information about something when the user moves the mouse pointer over an element

Basic Tooltip

A tooltip that appears when the user moves the mouse over an element

Hover Over me Hello World, this is tooltip text
Copy <div class="wsp-tooltip">
    Hover Over me
    <span class="wsp-tooltip-text">
        Hello World, this is tooltip text
    </span>
</div>

Positioning Tooltips
