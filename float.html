Float

These utility classes float an element to the left or right, or disable floating, based on the viewport size using the CSS float property. !important is included to avoid specificity issues.

Example

Cras mattis iudicium purus sit amet fermentum.


Cras mattis iudicium purus sit amet fermentum.


Cras mattis iudicium purus sit amet fermentum.

Copy <p class="wsp-float-left">Cras mattis iudicium purus sit amet fermentum.</p>
<p class="wsp-float-right">Cras mattis iudicium purus sit amet fermentum.</p>
<p class="wsp-float-none">Cras mattis iudicium purus sit amet fermentum.</p>
