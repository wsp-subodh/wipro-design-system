Slider

Slider method is used to create a selected element as slider in the web page. We can make slider as either horizontal or vertical.

Default Slider

100
Copy <div class="range-slider horizontal">
  <input class="range-slider__range" type="range" value="100" min="0" max="500">
  <span class="range-slider__value">100</span>
</div>

Verical Slider

100
Copy <div class="range-slider vertical">
  <input class="range-slider__range" type="range" value="100" min="0" max="500">
  <span class="range-slider__value">100</span>
</div>

Dual Range Slider

Copy <div class="dual-range" data-min="20" data-max="70">
  <span class="handle left"></span>
  <span class="highlight"></span>
  <span class="handle right"></span>
</div>
