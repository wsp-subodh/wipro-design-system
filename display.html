Hiding elements

For faster mobile-friendly development, use responsive display classes for showing and hiding elements by device. Avoid creating entirely different versions of the same site, instead hide element responsively for each screen size.

To hide elements simply use the .wsp-d-none class or one of the .wsp-d-{sm,md,lg,xl}-none classes for any responsive screen variation.

To show an element only on a given interval of screen sizes you can combine one .wsp-d-*-none class with a .wsp-d-*-* class, for example .wsp-d-none .wsp-d-md-block .wsp-d-xl-none will hide the element for all screen sizes except on medium and large devices.
