WSP Design System

Select

Using the contextual .wsp-form-* classes on an <select> element quickly provide contextual select.

Form controls

Textual form controls—like <select>s, are styled with the .wsp-form-control class. Included are styles for general appearance, focus state, sizing, and more.

Copy <form>
  <div class="wsp-form-group">
    <label for="exampleFormControlSelect1">Example select</label>
    <select class="wsp-form-control" id="exampleFormControlSelect1">
      <option>1</option>
      <option>2</option>
      <option>3</option>
      <option>4</option>
      <option>5</option>
    </select>
  </div>
</form>

Sizing

Set heights using classes like .wsp-form-control-lg and .wsp-form-control-sm .

Copy <select class="wsp-form-control wsp-form-control-lg">
  <option>Large select</option>
</select>

<select class="wsp-form-control">
  <option>Default select</option>
</select>

<select class="wsp-form-control wsp-form-control-sm">
  <option>Small select</option>
</select>

Disabled

Add the Disabled boolean attribute on an input to prevent modification of the input’s value.

Copy <div class="wsp-form-group">
  <label for="exampleFormControlSelect1">Example select</label>
  <select class="wsp-form-control" id="exampleFormControlSelect1" disabled>
    <option>1</option>
    <option>2</option>
    <option>3</option>
    <option>4</option>
    <option>5</option>
  </select>
</div>

Select Varient

you can indicate invalid and valid form fields with .is-invalid and .is-valid.

Copy   <select class="wsp-form-control is-valid">
    <option>1</option>
    <option>2</option>
    <option>3</option>
    <option>4</option>
    <option>5</option>
  </select>

  <select class="wsp-form-control is-invalid">
    <option>1</option>
    <option>2</option>
    <option>3</option>
    <option>4</option>
    <option>5</option>
  </select>
