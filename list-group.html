List group

The most basic list group is an unordered list with list items and the proper classes. Build upon it with the options that follow, or with your own CSS as needed.

Basic example

The most basic list group is an unordered list with list items and the proper classes. Build upon it with the options that follow, or with your own CSS as needed.

  • First Item
  • Second Item
  • Third Item
  • Fourth item
Copy <ul class="wsp-list-group">
    <li class="wsp-list-group-item">First Item</li>
    <li class="wsp-list-group-item">Second Item </li>
    <li class="wsp-list-group-item">Third Item </li>
    <li class="wsp-list-group-item">Fourth item</li>
</ul>

Active items

Add .active to a .list-group-item to indicate the current active selection.

</ul>
